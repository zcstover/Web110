HTML ELEMENTS

Tag Description Block, Inline, N/A Container or Empty
<--..--> An HTML comment tag. Used to Store comments, visible only in the HTML source. N/A Container
<!DOCTYPE> Lets the browser know what type of document to expect N/A Empty
<a> An a tag is for anchoring a link to another page or file Inline Container
<article> The <article> tag specifies independent, self-contained content. Block Container
<aside> The <Aside> defines some content aside from the content it is placed in. Block Container
<body> The <body> tag defines the document's body. Block Container
<br> The <br> tag inserts a single line break. N/A Empty
<dd> The <dd> tag is used to describe a term/name in a description list. Inline Container
<div> The <div> tag defines a division or a section in an HTML document. Block Container
<dl> The <dl> tag defines a description list. Block Container
<dt> The <dt> tag defines a term/name in a description list. Inline Container
<em> The <em> tag is a phrase tag. It renders as emphasized text. Inline Container
<fieldset> The <fieldset> tag is used to group related elements in a form. Block Container
<figcaption> The <figcaption> tag defines a caption for a <figure> element. Inline Container
<figure> The <figure> tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. Block Container
<footer> The <footer> tag defines a footer for a document or section. Block Container
<form> The <form> tag is used to create an HTML form for user input. Block Container
<h1-h6> The <h1> to <h6> tags are used to define HTML headings. Inline Container
<head> The <head> element is a container for all the head elements. Block Container
<header> The <header> element represents a container for introductory content or a set of navigational links. Block Container
<html> The <html> tag tells the browser that this is an HTML document. Block Container
<iframe> The <iframe> tag specifies an inline frame. Inline Container
<img> The <img> tag defines an image in an HTML page. Inline Empty
<input> The <input> tag specifies an input field where the user can enter data. Inline Empty
<label> The <label> tag defines a label for a <button>, <input>, <meter>, <output>, <progress>, <select>, or <textarea> element. Inline Container
<li> The <li> tag defines a list item. Inline Container
<link> The <link> tag defines a link between a document and an external resource. Inline Empty
<meta> The <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. N/A Empty
<nav> The <nav> tag defines a set of navigation links. Block Container
<ol> The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical. Block Container
<p> The <p> tag defines a paragraph. Inline Container
<script> The <script> tag is used to define a client-side script (JavaScript). N/A Container
<section> The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document. Block Container
<strong> The <strong> tag is a phrase tag. It defines important text. Inline Container
<style> The <style> tag is used to define style information for an HTML document. N/A Container
<table> The <table> tag defines an HTML table. Block Container
<td> The <td> tag defines a standard cell in an HTML table. Inline Container
<th> The <th> tag defines a header cell in an HTML table. Inline Container
<title> The <title> tag is required in all HTML documents and it defines the title of the document. Inline Container
<tr> The <tr> tag defines a row in an HTML table. Block Container
<ul> The <ul> tag defines an unordered (bulleted) list. Block Container
